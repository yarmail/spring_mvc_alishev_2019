Урок 35 Отношение Один к Одному (One to One)

(см картинку)

Выражение:
У гражданина страны может быть только
один паспорт этой страны.
Паспорт страны может быть только
у одного гражданина

Отношение
Parent(one) - Person
Child(one) - Passport
Человек может существовать без паспорта
Паспорт не может существовать без человека


Примеры:
Гражданин - Паспорт
Президент - Страна
Директор - Школа

Структурно:
Мы внедряем person_id в таблицу Passport
где он выполняет 2 роли: является Primary key
и foreign_key
