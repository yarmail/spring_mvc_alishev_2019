Урок 29 Конфигурация БД из внешнего файла

Если в рабочих файлах мы открыто указываем
логин и пароль, то любой человек может
это получить

Будем менять SpringConfig из урока 16
убирать логины и пароли из метода
dataSource()

---
Примечания
Желательно не передавать этот файл в хранилище
Например вставить в .gitignore
Вместо него показвать database.properties.origin
в котором показвать ключи без значений

При использовании ключа username после включения
Environment выдается ошибка 500 (вероятно цепляет другого пользователя
одно из решений - использовать user_name вместо username
