43 Выпадающие списки (select и option)

Для использования выпадающих списков применяют
теги, select и option
Пример:
<select id="id" name="name">
    <option value="0">0</option>
    <option value="1">1</option>
    <option value="2">2</option>
    ....
</select>
которые помещаются внутри формы
- на сервер отправляется value="...."

Для реализации этой задачи мы будем использовать
Thymeleaf, который позволяет соединить
модель и представление

Практика
Практику проводим в уроке 21
Задача, из общего списка назначить администратора
Это мы делаем на странице /admin,
на которой будет выпадющий список,
из которого мы выбираем одного человека и делаем его админом
Для этого задания мы делаем новый контроллер
AdminController
Получая id человека мы ему назначаем в БД значение
админ (да,нет)
